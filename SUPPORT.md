# Support

Please report issues through GitHub Issues.

Useful information to include:

- VS Code or Cursor version
- Operating system
- PaperMirror version
- Provider preset
- Base URL, without API key
- Model name
- Whether the issue happens on a small `.tex` example
- Relevant PaperMirror logs, with private content removed

Do not paste API keys or unpublished paper content into public issues.
